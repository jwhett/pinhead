### Pinhead

Requires the following environment variables:

- `TOKEN`: bot token
- `MAX`: maximum number of :pushpin: to pin a message
- `ROLE`: the name of the "moderator" role in the server
- `TLDR`: ID of the channel you want "logs" to go
